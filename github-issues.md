## Issue 4

**Title:** `AI coaching engine — signal detection logic + cue generation`

**Description:**
```
## What to build
The core AI logic that reads the live transcript and decides when and what to whisper to the user.

## Acceptance criteria
- [ ] Silence gap > 4 seconds triggers cue: "Don't fill this silence — wait."
- [ ] Trigger words detected ("always", "never", "final offer", "budget", "that's the best I can do") fire a context-aware counter prompt
- [ ] Concession pattern detected (user drops their number) triggers: "You've given ground — ask for something back."
- [ ] Cues are generated using Gemini Flash 2.5
- [ ] Each cue fired is saved to `cues_fired` in the session record

## Notes
Cues should be short — max 12 words. Use the session context (situation type, target number) loaded from setup to personalize the prompts.
```

## Issue 5

**Title:** `Whisper overlay UI — coaching cue display during live session`

**Description:**
```
## What to build
A subtle on-screen overlay that displays coaching cues during the live session without disrupting the transcript view.

## Acceptance criteria
- [ ] Cue appears as a small card/bubble at the top or bottom of the screen
- [ ] Cue auto-dismisses after 6 seconds
- [ ] New cue replaces old one if fired while one is already showing
- [ ] Distinct visual style from the transcript (different color, subtle border)
- [ ] Optional: device vibration on cue arrival (haptic feedback)

## Notes
Design should feel calm and non-alarming — this appears during a stressful moment. Avoid red colors or urgent-looking UI.
```

## Issue 6

**Title:** `Post-session debrief screen — transcript summary, cues, AI insight`

**Description:**
```
## What to build
A summary screen shown after the user ends a session, giving them a recap of how the negotiation went.

## Acceptance criteria
- [ ] Shows full session transcript (scrollable)
- [ ] Shows total cues fired + list of each cue with timestamp
- [ ] Shows talk ratio (% of words spoken by user vs counterpart)
- [ ] Shows one AI-generated insight about the session (e.g. "You held your anchor well but filled 3 silences — practice pausing.")
- [ ] Session is saved and accessible in history

## Notes
AI insight generated by Gemini Flash 2.5 using the full transcript + cues fired as context.
```
